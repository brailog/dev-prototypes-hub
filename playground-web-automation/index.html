Playground para Automação v2

Bem-vindo ao Playground de Automação!

Esta página é um ambiente interativo projetado para praticar e testar scripts de automação web. Você pode usar ferramentas como PyAutoTk, Selenium e Playwright para interagir com os elementos em cada seção.

Clique em "Começar" para iniciar. Após iniciar, use os botões "Anterior" e "Próximo" que aparecerão no canto superior para navegar entre os desafios. A seção de informações abaixo fornecerá detalhes e exemplos de código para cada tarefa.

Para saber mais sobre o PyAutoTk, confira os links: GitHub, PyPI, e a Documentação Oficial.
